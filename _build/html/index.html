Welcome to image2character’s documentation!¶

Code is also in github

Program Flow:¶

  1. from an image by haar cascade get plate,
  2. then from plate get regions (by MSER) ,
  3. from regions get letters/digits by SVM/logistic regression

Usage:¶

python3 image2characters.py filename

More Description:¶

  • The flow of the program is defined in image2characters.py.

  • It uses modules:

    1. rekkariDetectionSave.py
    - to find a region in the image containing the plate
    (haar cascades of opencv)

    2. filterImage.py
    - to make the image more clear

    3. filterCharacterRegions.py
    (inherits from initialCharacterRegions.py)
    - to get list of six-rectangles for possible plete-character regions of the image

    4. myClassifier.py
    - to detect characters
